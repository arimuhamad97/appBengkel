üöÄ 5 Cara Mudah Akses

1. Bookmark Browser Paling Mudah

  1. Buka aplikasi di browser
  2. Tekan Ctrl+D atau klik ikon bintang (‚òÖ)
  3. Simpan di Bookmarks Bar
  4. Klik bookmark untuk akses cepat

2. Shortcut Desktop Sangat Praktis

  1. Double-click file: buat-shortcut-client.bat
  2. Shortcut akan muncul di Desktop
  3. Klik shortcut untuk membuka aplikasi

3. Domain Lokal Paling Profesional

  1. Klik kanan: setup-domain-lokal.bat
  2. Pilih "Run as administrator"
  3. Akses dengan: http://bengkel.local:5173

4. Homepage Browser Auto Buka

  1. Buka Settings browser (Ctrl+,)
  2. Cari "On startup"
  3. Pilih "Open a specific page"
  4. Masukkan URL aplikasi

5. Kiosk Mode Untuk Kasir

  1. Double-click: buka-bengkel-fullscreen.bat
  2. Aplikasi buka fullscreen
  3. Copy file ke Startup untuk auto-start
